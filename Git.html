Spis podstawowych komend do git-a.



Poszukałem w polskim internecie i jest mnustwo bezwartościowych wpisów blogowych o tym, czym jest git ale nie było zbioru komend wraz z funkcją jaką te komendy wywołują, więc postanowiłem w tym wpisie maxymalnie upakować wiedzę na temat komend używanych w git-cie, do których będę mógł sobie wracać gdybym zapomniał ich składni.


git add = dodaj całą ścieżkę do zmodifikowane pliku

git log = pokazuje repozytoria

git status = pokazuje, czy zmiany zostały zapisane

git checkout = sprawdza jakie zmiany zaszły w wersjach naszych repozytoriów, które elementy się zmieniły po zapisie. Na końcu tej komenty wklej początek kodu dwóch wersji jakie chcesz sprawdzić. A gdy skopiujesz tylko jeden kod to przywrucisz jego wersję
